# Landing Page

Responsive static landing-page project with bundled assets, service-worker files, and a portfolio/agency-style layout.

## Scope

- Static HTML landing page with hero, services, portfolio, team, and review-style sections.
- CSS and JavaScript assets organized under `assets/`.
- Bootstrap, jQuery, Slick, Shuffle, and supporting UI libraries.
- Service worker and cache manifest included from the original build.

## Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery

## Project Structure

```text
.
|-- index.html
|-- assets/css/
|-- assets/js/
|-- assets/img/
|-- service-worker.js
`-- cache.manifest
```

## Run Locally

Open `index.html` directly in a browser or serve the directory with a local static server.
